# 🔐 Quantum Password Generator

A beginner-friendly Quantum Computing project built using **Qiskit** and **Streamlit** that generates secure passwords using quantum randomness.

---

## 📌 Project Overview

Traditional password generators rely on classical pseudo-random algorithms.

This project leverages **Quantum Computing concepts** such as:

* Qubits
* Superposition
* Hadamard Gates
* Quantum Measurement

to generate highly random and secure passwords.

---

## 🚀 Features

* Generate secure passwords using quantum randomness
* Custom password length selection
* Includes:

  * Uppercase Letters
  * Lowercase Letters
  * Numbers
  * Special Characters
* Password Strength Analysis
* Interactive Streamlit UI

---

## 🧠 Quantum Concepts Used

### Qubits

Basic unit of quantum information.

### Superposition

Allows qubits to exist in multiple states simultaneously.

### Hadamard Gate

Creates superposition in a quantum circuit.

### Measurement

Collapses the qubit into a random classical state.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/mohanveeramanikantak/quantum-password-generator.git
cd quantum-password-generator
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Application

```bash
streamlit run app.py
```
